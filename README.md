The ALE environment part was based on :
<ol>
<li> DQN : https://github.com/devsisters/DQN-tensorflow </li>
</ol>

<h2>Requirements</h2>
- Tensorflow
- numpy
- scipy
- scikit-image
- ale_python_interface : install from <a href="https://github.com/mgbellemare/Arcade-Learning-Environment">here</a>
